# Ecommerce KPI Report Automation (Python)

## Overview
This project automates the generation of recurring ecommerce KPI reports using Python. The script processes raw transactional data, calculates key performance metrics, and exports structured, formatted Excel reports for business analysis.

## Business Problem
Operations and analytics teams often spend time manually preparing recurring performance reports. This automation reduces manual effort, improves consistency, and enables faster access to insights.

## Requirements
- pandas
- xlsxwriter

## Tools & Technologies
- Python
- Excel

## KPIs Generated
- Revenue
- Orders
- Units Sold
- Unique Customers
- Average Order Value (AOV)
- Return Rate
- Churn Rate

## Output
The script generates a multi-sheet Excel file including:
- Summary KPIs
- Daily performance metrics
- Monthly performance metrics
- Category-level performance
